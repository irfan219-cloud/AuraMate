# ⚡ AuraMate Quick Reference Card

## 🚀 Start App
```bash
npm install
npm run dev
```

## 🎙️ Voice Features

### Enable Voice
- Use Chrome or Edge browser
- Allow microphone permissions
- Click mic button in chat

### Test Voice
1. Go to Chat page
2. Click microphone 🎤
3. Say: "I'm feeling stressed"
4. Watch visualizer animate
5. Hear AI response

## 🐛 Troubleshooting

### Voice Not Working?
- Check browser (Chrome/Edge)
- Allow mic permissions
- Check mic not muted
- Fallback: Type instead

### App Not Starting?
```bash
rm -rf node_modules
npm install
npm run dev
```
